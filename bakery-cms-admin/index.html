Developer Setup (Action Required)

To make Google Login work, you must set up your Client ID and API Key.

  1. Go to Google Cloud Console.
  2. Create an OAuth Client ID (Web Application).
  3. Add your Vercel URL to Authorized JavaScript origins.
  4. Open this `index.html` file in a text editor and paste your Client ID at the top of the app.js file!
