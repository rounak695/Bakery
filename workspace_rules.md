# Cinematic Design Rules

This project strictly follows a sleek, minimal, and Cinematic aesthetic. When styling elements, adhere to the following rules:

1. **Minimalist UI**: UI elements should be unobtrusive and elegant. Avoid bulky or "cartoonish" components.
2. **Color Palette**: Rely on deep dark tones (blacks, dark grays) and crisp whites to create high contrast. Avoid harsh primary colors like bright red or yellow unless used very sparingly as accents.
3. **Typography**: Use elegant, refined typography. The brand logo should use a premium flowing script font (e.g., Great Vibes), while UI text should use clean, modern geometric or neo-grotesque sans-serifs.
4. **Cinematic Presentation**: Video content is the focal point. Ensure videos span beautifully across the layout without heavy, distracting borders. Use subtle gradients or dark overlays to ensure text readability over media.
5. **Subtle Animations**: Interactions should feature smooth, eased transitions (e.g., fades, gentle scaling). Avoid aggressive or bouncy animations.
